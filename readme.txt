[FR]
Projet Java Interface graphique - version 1
14 mars 2013  
CF : représentation uml dans ce même dossier

Visualisation et manipulation de figures géométriques

Le but de la premiere version est de concevoir un programme initial 
sans aucun composant graphique interactif (clic de souris...). 

Il s'agit d'établir les classes représentant les différentes formes géométriques.
On affiche un JFrame et un menu en console permet d'effectuer les actions suivantes :
- Créer une figure géométrique
	- quadrilatère quelconque
	- Rectangle
	- Triangle
	- Cercle
	
- Changer la couleur de l'objet (noir par défaut)

- Effectuer une translation 

- Modifier la forme et l'emplacement en redéfinissant les points de saisie

Java project : graphic interface - version 1
14 March 2013 
SEE : uml modelisation in this same repo

Visualizing and manipulating geometrical figures 

The goal of this first version is to develop an initial program
without any graphic and interactive component for now (mouse clics ...) 

The v1 is about classes and setteling the different geometrical shapes.
A JFrame is displayed and the user can, by interacting with the menu in the console,
- Create a geometrical figure
	- polygon 
	- Rectangle
	- Triangle
	- Cercle

- Change the color's object 

- Make a translation 

- Change the shape and its place in the frame by redefining the grasp points
